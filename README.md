# Chinook.SQL
Mastering SQL with the Chinook database
This week, I strengthened my SQL skills by working with the Chinook database and solving real business problems using SQL Server.
Some of the concepts I practiced include:
SQL queries (SELECT, WHERE, ORDER BY)
Aggregate functions (COUNT, SUM, AVG)
GROUP BY and HAVING
INNER JOIN, LEFT JOIN, and RIGHT JOIN
Subqueries
Window functions (ROW_NUMBER(), RANK(), DENSE_RANK(), PARTITION BY)
I also answered business questions such as:
Identifying the top customers by total spending.
Finding the customer with the highest number of purchases.
Determining the country that generated the highest revenue.
Analyzing sales data to gain business insights.
Each exercise is helping me become more confident in writing SQL queries and solving real-world business problems.
